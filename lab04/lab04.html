Heading and Paragraphs

Heading and paragraphs are BLOCK elements

Heading

There are 6 levels of heading. Headings are logical marked. Do not skip levels. If you want to change the size of a heading, don't change its level - change the CSS.

Paragraph

Paragraph contain the body text of a web page. You cannot put another block element inside a paragraph.

Special Characters

Whiel HTML support many name special character like ®(#174), XML only supports 5 namely: " ' & < and >. So to keep your HTML compliant you need to use numbers for all the others.
